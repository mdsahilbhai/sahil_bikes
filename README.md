# 🏍️ My Lovely Bikes

A simple and responsive website showcasing **Super Bikes** and **Sports Bikes** along with their **full specifications and price in INR**.

---

## 📌 Project Overview

**My Lovely Bikes** is a front-end web project built using **HTML, CSS, and JavaScript**.
It displays a collection of high-performance bikes with details like engine capacity, power, top speed, and pricing.

This project is ideal for:

* Beginners learning web development
* Frontend portfolio projects
* DevOps deployment practice (GitHub Pages, Netlify, Vercel)

---

## 🚀 Features

* 🏍️ List of Super Bikes & Sports Bikes
* 📊 Detailed specifications:

  * Engine (CC)
  * Power (BHP)
  * Top Speed (km/h)
* 💰 Price displayed in INR
* 🎨 Responsive and modern UI
* ⚡ Dynamic content using JavaScript

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)
